# NotaeShell

**Notaesh** - это простой менеджер заметок и задач в вашем терминале. Написан на bash. Лёгок в интеграции и конфигурации.

## Установка

```
git clone https://codeberg.org/Itcor/NotaeShell.git
cd NotaeShell
chmod +x install.sh
./install.sh
```


## Использование

| Команда        | Описание                              |
|----------------|---------------------------------------|
| `nsh`          | Вывод всех заметок                    |
| `nsh -h`       | Выводит список всех команд            |
| `nsh <text>`   | Добавляет новую заметку               |
| `nsh -d [num]` | Удаляет заметку с указанным номером   |
