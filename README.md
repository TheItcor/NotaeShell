# NotaeShell

**Notaesh** - A humble notes and tasks manager in your terminal. Written in bash. It is easy to integrate and configure.

## Install

```
git clone https://codeberg.org/Itcor/NotaeShell.git
cd NotaeShell
chmod +x install.sh
./install.sh
```


## How to use?

| Command        | Description                          |
|----------------|--------------------------------------|
| `nsh`          | Prints all notes                     |
| `nsh -h`       | Prints all commands                  |
| `nsh <text>`   | Adds a new note                      |
| `nsh -d [num]` | Deletes the selected note            |
